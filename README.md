# Portail du Lycée Technique Dan Kassawa (LTDK) de Maradi

Ce projet est le site web officiel et la plateforme communautaire pour le Lycée Technique Dan Kassawa (LTDK) de Maradi, au Niger. Développé avec Next.js, Firebase et Tailwind CSS, il sert de hub central pour les élèves actuels, les anciens élèves (l'Amicale), l'administration et le grand public.

## Fonctionnalités Principales

*   **Portail Dynamique** : Contenu géré via la base de données Firestore, permettant des mises à jour faciles des actualités, des membres de l'administration, des portraits d'anciens, etc.
*   **Espace Élèves** : Un fil d'actualités pour les associations étudiantes (CJM, AJP) et un tableau des opportunités (bourses, stages, concours).
*   **Section Amicale** : Un espace dédié aux anciens élèves avec des actualités, des portraits et un programme de mentorat.
*   **Formulaires Interactifs** : Des formulaires fonctionnels pour rejoindre l'Amicale, devenir mentor ou trouver un mentor, avec sauvegarde des demandes dans Firestore.
*   **Bibliothèque Numérique** : Une banque de documents (sujets de Bac, cours, exercices) avec un système de recherche et de téléversement de fichiers via Firebase Storage.
*   **Authentification** : Un système de connexion sécurisé via Firebase Authentication (Google) pour accéder aux contenus réservés aux membres, comme l'annuaire.
*   **Design Moderne et Responsive** : Une interface utilisateur soignée construite avec ShadCN UI et Tailwind CSS, entièrement adaptable aux mobiles et aux ordinateurs.

## Stack Technique

*   **Framework Frontend** : [Next.js](https://nextjs.org/) (avec App Router)
*   **Backend & Base de Données** : [Firebase](https://firebase.google.com/) (Firestore, Firebase Authentication, Firebase Storage)
*   **Styling** : [Tailwind CSS](https://tailwindcss.com/)
*   **Composants UI** : [ShadCN UI](https://ui.shadcn.com/)
*   **Gestion de formulaires** : [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)

Ce projet a été initialement développé avec l'aide de l'IA de Firebase Studio.
